# rest-api-test

Projeto criado para estudar como desenvolver uma API Rest.
Para testar a API basta rodar os comandos abaixo:

```
npm install
npm start
```

## rotas

- __GET ALL__ = '/cadastros'
- __GET BY ID__ = '/cadastro/:id'
- __POST__ = '/cadastro'
- __PUT__ = '/cadastro/:id'
- __DELETE__ = '/cadastro/:id'
